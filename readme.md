# 一、學習目標:
在現今生成式 AI（Generative AI）蓬勃發展的時代，還是有許多人雖熟練使用ChatGPT等工具， 但卻不了解其背後的基礎原理。
因此本作業旨在透過實作傳統方法NTP方法，理解文本處理的數學基礎，並同樣使用現代AI完成相同任務，去比較兩種方法，培養選擇適當工具的判斷力。

# 二、傳統方式實作:

A-1：TF-IDF 文本相似度計算
- 手動計算TF-IDF。
- 使用scikit-learn內建方法進行驗證。
- 計算文本相似度
- 視覺化相似度矩陣

A-2：文本分類
- 使用jieba進行中文分詞。
- 使用stopwordsiso移除停用詞。
- 比較分詞前後的TF-IDF向量差異。

A-3：自動化摘要
- 利用matplotlib或seaborn視覺化關鍵字權重分佈。
- 撰寫分析文字，說明哪些詞對相似度的影響最大。

# 三、AI方法實作

B-1:語意相似度計算
- 利用OpenAI API的prompt進行文本摘要與語意相似度分析。

B-2：文本分類
- 利用OpenAI API的prompt進行文本對比

B-3:自動話摘要
- 利用OpenAI API的prompt進行文章摘要

請先安裝以下套件：
pip install numpy pandas matplotlib seaborn scikit-learn jieba stopwordsiso openai tqdm
點擊全部執行即可得到輸出結果!!!


# 四、結果與討論:

傳統方法成果
- TF-IDF 熱圖清楚顯示文本間的相似度。
- 分詞與停用詞處理顯著提升結果精確度。
- 可明確觀察各詞彙對文本相似度的貢獻。

AI方法成果
- 模型能識別語義關聯，輸出更貼近人類理解。
- 能自動摘要、歸納主題，超越字面分析。
- 但生成結果具有隨機性，且可重現性不高。

綜合比較
- 傳統 NLP 方法注重「結構化、可解釋性」，
- 而現代 AI 模型強調「語意、泛化能力」。
- 兩者並非取代關係，而是互補並存。

心得與反思:
透過這次作業，我更深刻理解NLP的數學邏輯與資訊理論基礎，也得知能處理更高層次的語意關聯，讓文本分析更貼近人類思考。不過也必須得注意AI模型雖強大，但若不理解底層邏輯，將無法評估其可靠性。
